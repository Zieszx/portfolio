# Ieskandar Zulqarnain - Portfolio

A modern, responsive portfolio website built with React.js and Vite, showcasing my skills, projects, and professional experience as a Software Developer and Engineer.

## üåü Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations and interactions
- **Interactive Sections**:
  - Hero section with animated typing effect
  - About section with personal information and stats
  - Skills section with animated progress bars
  - Portfolio section with project filtering
  - Resume/CV section with work experience and education
  - Achievements section highlighting academic and professional accomplishments
  - Contact section with contact information and form

## üöÄ Technologies Used

- **Frontend**: React.js, JSX, Vite
- **Styling**: CSS3, Bootstrap 5 (via CDN)
- **Icons**: Bootstrap Icons
- **Animations**: AOS (Animate On Scroll)
- **Other Libraries**:
  - Typed.js for typing animations
  - Isotope.js for portfolio filtering
  - GLightbox for image lightbox
  - PureCounter for animated counters
  - Swiper.js for sliders

## üîß Setup and Installation

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Run development server**
   ```bash
   npm run dev
   ```

3. **Build for production**
   ```bash
   npm run build
   ```

4. **Preview production build**
   ```bash
   npm run preview
   ```
